# Version Time Machine

A simple web tool for moderators, compliance teams, and security auditors to check **what was the latest safe/real version** of popular software at any given date in the past.

Enter any date → instantly see the most recent **stable** versions of major platforms that were available **at that time** (with a safety buffer of +14 days).

Useful for:
- Checking historical compliance (e.g. "Was this browser version latest on Jan 1, 2026?")

## Features

- Pick any date (default: today minus safety buffer)
- Shows latest **Stable** versions as of that date for:
  - Browsers (Chrome, Edge, Firefox, Safari)
  - macOS
  - iOS
  - iPadOS
  - VMware ESXi
  - .NET
  - M365

Browse Category wise to see EOL and Release Notes from a bunch of vendors
  - Browsers
  - Operating Systems
  - Business AV
  - Servers
  - Firewalls
  - AWS
  - Software Support
